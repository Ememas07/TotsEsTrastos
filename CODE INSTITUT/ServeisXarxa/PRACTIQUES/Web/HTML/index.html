EXPLICACIÓ DE LA PRÀCTICA 1

Per la pràctica 1 ens hem connectat al servidor de gimensia amb filezilla i amb CMD

FASES

  • Instal·lació FileZilla
  • Configuració de sitios
  • Creació de carpetes i arxius
  • Permisos
  • CMD

FASE 1: Instal·lació FileZilla

  • Per començar hem anat a la pàgina de filezilla i l'hem instal·lat

FASE 2: Configuració de sitios

  • Anam a la configuració de sitios i configuram:
    • Protocol: FTP
    • Host: FTP.Gimnesia.net
    • Port: 21
    • User: Alumne.gimnesia.net
    • Password: entradaFTP21
  • I pitjam connect

FASE 3: Creació de carpetes i arxius

  • Després hem creat totes les carpetes i arxius que tu has dit al servidor

FASE 4: Permisos

  • També varem configurar els permisos dels arxius

FASE 5: CMD

  • Per connectar-nos per cmd posam "ftp ftp.gimnesia.net"
  • Per fer un directori posam mkdir

EXPLICACIÓ DE LA PRÀCTICA 2

Per aquesta pràctica hem configurat el nostre server de filezilla

FASES

  • Instal·lació FileZilla server
  • Configuració inicial
  • Connexió al servidor
  • Permisos de grups, usuaris
  • Proves de connexió

FASE 1: Instal·lació FileZilla server

  • Anam a la pàgina oficial de filezilla server i la descarregam

FASE 2: Configuració inicial

  • Feim el configurador inicial
  • Pitjam connect to server i posam usuari, contrassenya i port d'administració

FASE 3: Proves de connexió

  • Primer provam de fer un ping a la màquina
  • Si funciona, també provam un tracert
  • Finalment provam de fer "ftp" i la IP de la màquina

FASE 4: Connexió al servidor

  • Anam a un altre màquina i posam la seva IP al filezilla client, posam un usuari i contrassenya i toca funcionar.

FASE 5: Permisos de grups, usuaris

  • Per administrar permisos de grups i usuaris se fa tot desde la part de "users" i "groups" del filezilla server

EXPLICACIÓ DE LA PRÀCTICA 3

Per aquesta pràctica hem creat un servidor apache2 amb vagrant i virtualbox

FASES

  • Instal·lació virtualbox i vagrant
  • Inicialització i preparació vagrant
  • Arrancam i entram a la màquina
  • Instal·lam apache
  • Arrancam apache
  • Comprovam que està funcional

FASE 1: Instal·lació Virtualbox i Vagrant

  • Per començar hem d’instalar el vagrant i el virtualbox versió 7.0
  • Anirem a la pàgina de descàrregues del vagrant i ens descarregarem la darrera versió.

FASE 2: Inicialització i preparació Vagrant

  • Després crearem una carpeta amb un nom que volguem (en el meu cas posarem apache-vm), i escriurem “vagrant init”
  • Això ens crearà un vagrantfile, i aquest vagrantfile nosaltres l’editarem.
  • Canviarem la linea de "config.vm.box" a "bento/ubuntu-22.04
  • Descomentarem les linees de:
    • config.vm.network "forwarded_port", guest: 80, host: 8080
    • config.vm.network "public_network", ip: "10.0.100.161"
    • config.vm.network "public_network"

FASE 3: Arrancam i entram a la màquina

  • Feim un vagrant up
  • I després un vagrant ssh

FASE 4: Instal·lam i arrancam apache

  • Abans de res, farem un "sudo apt-get update" per actualitzar els paquets
  • Després, "sudo apt-get install apache2" per instalar el servidor de apache
  • finalment farem "sudo service apache2 start" per arrancar-lo

FASE 5: Comprovam que funciona

  • Si feim "sudo service apache2 status" podem comprovar si el server de apache funciona.
  • Podem provar de fer "curl localhost" per comprovar que realment funciona
  • Després podem fer un "IP A" per veure la IP de la màquina
  • I si introduim aquesta IP de la màquina a un navegador tocam poder vuere que l'apache2 s'ha instal·lat i funciona correctament
